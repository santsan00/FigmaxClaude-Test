## Welcome

This is the first post on this blog. It's built with **Next.js 15** and uses Markdown files as the content source — no database, no CMS.

Each post is a `.md` file in the `/posts` directory. The frontmatter (the section between `---`) holds the title, date, and excerpt. Everything below is rendered as HTML.

### Why Markdown?

- It's version-controlled alongside the code
- It's easy to write
- It's easy to read as plain text

### What's next?

This project is a test of the **Figma ↔ Claude Code MCP** integration. The plan is to design the blog UI in Figma, then use the MCP server to pull those design tokens and component specs directly into the code.

Stay tuned.
